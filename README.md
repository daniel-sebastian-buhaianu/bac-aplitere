# bac-aplitere
Scrieti un program C/C++ care citeste de la tastaturaun sir de caractere format din cel mult 100 de caractere (litere mici, litere mari ale alfabetului englez si spatii) si afiseaza pe ecran litera care apare de cele mai multe ori si numarul ei de aparitii, ca in exemplu. Daca sunt mai multe litere care apar de acelasi numar maxim de ori se vor afisa toate.

| se citeste de la tastatura          | se afiseaza pe ecran |
|-------------------------------------|----------------------|
| Voi da Bacalaureatul la Informatica | a 8                  |
